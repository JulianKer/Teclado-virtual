# Virtual Keyboard
---
This is a virtual keyboard project. Some features include:
<ul>
<li>Uppercase letters.</li>
<li>Lowercase letters.</li>
<li>Dellete all button.</li>
<li>Delete button for individual characters.</li>
<li>Numbers.</li>
<li>Special characters.</li>
</ul>
